À propos du projet

Ce projet fut l'un des projets sur lesquels j'ai du travailler lors de mon stage chez Astek en fin de première année chez MyDigitalSchool. J'ai eu pour mission de refaire la page d'acceuil du site SFR avec WordPress afin que SFR puisse juger si WordPress est un CMS adapté pour la refonte de leur site internet. Malheureusement, il n'est pas possible de refaire cette page d'accueil en utilisant les thèmes proposés par WordPress car reproduire quelque chose d'aussi précis avec des thèmes prédéfinis. C'est pourquoi j'ai du faire un thème personalisé sur WordPress. Ces thèmes sont développés avec du HTML/CSS/JS classique pour faire les fondations du thème, mais il faut ajouter à cela du PHP implémenté directement dans le HTML, afin de rendre les éléments qui l'on souhaite modifiable dans le customizer de WordPress.

Objectifs du projet

  • Recréer l'interface utilisateur de SFR
  • Découvrir les thèmes personalisés de WordPress
  • Voir si WordPress peut être un CMS adapté pour refaire intégralement le site de SFR

Défis rencontrés

Le plus dur dans ce projet a été d'apprendre une nouvelle bibliothéque PHP car WordPress utilise par exemple des Hook et des API en PHP qui lui sont propre. Ce projet m'a non seulement permis de m'améliorer en HTML/CSS, mais m'a aussi apris une nouvelle manière de développer du PHP via WordPress, ce qui rendais ce projet très interéssant.

Frontend

HTML5, CSS3, customizer WordPress

Backend

PHP avec la doc de WordPress

Résultats et apprentissages

Ce projet m’a permis de renforcer mes compétences en HTML/CSS et de découvrir le développement de thèmes personnalisés sous WordPress. J’ai appris à utiliser le PHP spécifique à WordPress, notamment les hooks et le customizer, pour rendre les contenus dynamiques. Cela m’a aussi appris à m’adapter à de nouvelles contraintes techniques rapidement.
